# [`属性透传`](https://cn.vuejs.org/guide/components/attrs.html#class-and-style-merging)

- Attributes 继承

```html
<!-- <MyButton> 的模板 -->
<button class="btn">click me</button>
<!-- 使用 -->
<MyButton class="large" />
<!-- 渲染结果 -->
<button class="btn large">click me</button>
```

> `class` 就会合并

## 关闭继承

```html
<script setup>
defineOptions({
  inheritAttrs: false
})
// ...setup 逻辑
</script>
```
