# [`slot`](https://cn.vuejs.org/guide/components/slots.html)

> - 类似 `react` 的 `children`

## 基础使用

```html
<!-- FancyButton 组件 -->
<button class="fancy-btn">
  <slot></slot> <!-- 插槽出口 -->
</button>

<!-- 使用 -->
<FancyButton>
  Click me! <!-- 插槽内容 -->
</FancyButton>

<!-- 渲染 -->
<button class="fancy-btn">
  Click me!
</button>
```

## 默认值

```html
<button type="submit">
  <slot>
    Submit <!-- 默认内容 -->
  </slot>
</button>
```
